# ⚡ Calculadora de Consumo de Energia

## 📌 Objetivo
Este projeto calcula o consumo mensal de energia elétrica de um aparelho com base no uso diário.

## 🛠️ Tecnologias
- 🐍 Python
- 💻 VS Code
- 🌐 GitHub

## 📊 Fórmula utilizada
Consumo mensal (kWh):

(consumo = potência × horas por dia × 30) / 1000

## ▶️ Como executar

1. Clone o repositório:git clone LINK_DO_REPOSITORIO
2. Acesse a pasta:cd consumo-energia
3. Execute:python app.py
